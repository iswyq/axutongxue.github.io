告诉你一条野路子，实现天翼云上传不限速

发表于 2018-12-23 | 分类于 折腾软件 | | 阅读数

还是那个话题：“网盘”

最近百度云又进行了一波更新，当然是变得更加难用了——不登录无法下载文件。

导致的问题就是：如果用了破解百度云限速的软件来下载文件，容易被帐号限速

（短时间内非会员下载大量文件，易导致帐号被限速，虽然一般不下载过几天会自己解除）

于是阿虚在寻找国内网盘的替代品。

当然，你们从文章标题就能看出，最后我选择了天翼云。

一、天翼云优缺点

天翼云对我来说还是挺不错的：电信旗下服务，稳定、靠谱，不担心隐私问题！

优点

  • 不是一定要电信手机号才能注册

  • 大容量：免费提供2T网盘

  • 下载不限速

缺点

  • 分享的文件需要经过一段时间审核（阿虚不知道是否为人工审核，应该不是）
  • 分享的文件别人需要登录才能下载（尽管油猴有免登陆即可下载脚本）
  • 之前对我来说唯一缺点就是上传限速

不过这个问题，被阿虚后来找到路子解决了

二、尝试过的方法

很早之前就想着用天翼云，但无奈上传限速，不甘心的我琢磨许多办法，都不行

  • 试过从电信天翼开放平台上找路子，结果：失败

  • 因为189邮箱和天翼云是互通的，试过从189邮箱上传后转存到天翼云来实现上传不限速，结果：依然是会限速。

三、解决办法

依然是不甘心，到处寻找后，偶然发现天翼云旗下有一个软件：家庭云

  • 官网地址：http://home.cloud.189.cn/

  • 提供：Windows、Android、IOS客户端

然后阿虚一试，果不其然！

利用家庭云上传文件是不限速的。

（我在学校，自身网速不快，不过下面这个速度已经是学校宽带的上传上限了）

于是实现天翼云上传不限速的办法很简单：先上传到家庭云，再在家庭云客户端中，转存文件到天翼云即可。

备注：PC端登录，必须用手机端进行扫码登录

写这篇的原因有两点

  1. 是自己百度谷歌解除天翼云上传限速的方法无果（但谷歌不到的问题不代表就没办法解决了）

  2. 是最近发现分享的部分资源，比如PPT模板——分享的链接会间歇性失效。（大抵是访问人数过多引起的，过段时间可能会重新打包上传一份天翼云。）

现在想要分享舒心，别人下载舒心也挺麻烦的。

许多分享者是没有考虑到下载者的用户体验的。

最最起码，下载得不限速吧。

这也是我为什么不用百度云分享文件的原因。
